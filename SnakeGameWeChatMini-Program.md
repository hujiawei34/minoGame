# Snake Game WeChat Mini-Program

## Core Features

- Touch-based directional controls

- Real-time snake movement

- Food collection and scoring

- Collision detection system

- High score persistence

- Mobile-optimized UI

## Tech Stack

{
  "MiniProgram": "WeChat Mini-Program native framework with JavaScript/TypeScript, WXSS styling, Canvas API for rendering, and WeChat storage for persistence"
}

## Design

Modern minimalist design with dark green theme, neon green snake graphics, golden food items, touch-optimized controls, and clean typography optimized for mobile WeChat environment
